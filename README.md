World Time Buddy
================


Display timezones and times


Installation
------------

Install with

```
pip install https://github.com/serialdoom/world_time_buddy
```

Sample Configuration file
-------------------------

```
timezones:
  - America/Denver
  - America/New_York
  - America/Los_Angeles
```
